# Spring Boot Users & Cars API

API REST desarrollada con Spring Boot que permite crear usuarios y asignarles autos.
Proyecto pensado para practicar relaciones **OneToMany / ManyToOne** con **JPA + Hibernate**.

---

## 🛠 Tecnologías usadas

- Java 
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman

---

## 📌 Funcionalidades

### Usuarios
- Crear usuario
- Listar usuarios
- Buscar usuario por ID
- Actualizar usuario
- Eliminar usuario

### Autos
- Crear autos
- Asignar autos a un usuario
- Listar autos de un usuario

Relación:
- **Un usuario puede tener muchos autos**
- **Un auto pertenece a un usuario**
